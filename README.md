# Bon pour - D'ART'gil Café

Page de bon cadeau.

## Fichiers

- `index.html`: bon cadeau plein écran (sans scroll)
- `style.css`: styles principaux
- `qr.html`: page dédiée au QR code imprimable
- `assets/logo-dartgil-blanc.png`: logo utilisé

## Déploiement GitHub Pages

1. Pousser ce dossier sur un repo GitHub (par exemple `bonpour`).
2. Activer **Settings > Pages**.
3. Source: `Deploy from a branch`.
4. Branch: `main` et dossier `/ (root)`.
5. URL attendue: `https://<user>.github.io/<repo>/`.

## QR Code imprimable

1. Ouvrir `qr.html`.
2. Remplacer l'URL par l'URL GitHub Pages finale.
3. Cliquer `Mettre à jour`.
4. Cliquer `Imprimer`.
